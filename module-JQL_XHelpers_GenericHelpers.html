(inner) createRegExp(regexStr, flags) → {RegExp}

Source:
  • Helpers.js, line 62
A tagged template function to create a Regexp from multiple lines, with or without comments.
Example
createRegExp`
  ^[\p{L}]              //=> always start with a letter
  [\p{L}_\.#\-\d+~=!]+  //=> followed by letters including _ . # - 0-9 ~ = or !
  // [...]
  ${`gui`}              //=> flags ([g]lobal, case [i]nsensitive, [u]nicode)`
// ---
// Result: /^[\p{L}][\p{L}_\.#\-\d+~=!]+/giu
