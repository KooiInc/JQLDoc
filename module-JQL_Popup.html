JQL/Popup

a small library to create/use (modal) popups
Source:
  • Popup.js, line 1

Methods

(static) popupFactory($) → {Object}

Source:
  • Popup.js, line 20
This factory delivers 4 methods to create or remove a (modal) popup window in the browser.
  • create
  • createTimed
  • remove
  • removeModal
All popups are modal (the background page is inactivated until the popup is closed). A normal popup can be closed by clicking a small icon or outside the box. A 'really modal' popup can only be closed using a handler you define yourself.
